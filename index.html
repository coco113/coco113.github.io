Wie kreierst du eine Webseite?

Erster Schritt: Software installieren

Bevor du mit dem Erstellen einer Webseite beginnen kannst, musst du einige Programme auf deinem PC oder Mac installieren.

Klicke hier, um Python herunterzuladen. Achtung, du brauchst die neuste Version!

Nebst Python brauchst du andere wichtige Programme, die dir beim Erstellen der Webseite nützlich sein werden:

  • Git
  • Atom

Wenn du diese Programme erfolgreich installiert hast, musst du dich zudem auf der Plattform Github registrieren.
